# Optimization

本仓库主要实现南京航空航天大学人工智能学院李振华老师课`《最优化方法》`中涉及到的编程题目，仓库里的 `LectX` 与上课的顺序一致。

## 📌 项目简介  
- 该仓库用于实现《最优化方法》课程中的编程作业  
- 适用于学习最优化理论、算法和实践的同学  
- 包含不同优化方法的代码实现

## 🗂 目录结构
```text
.
```

## 🚀 如何使用
1. 克隆仓库：
   ```bash
   git clone https://github.com/myRan-ML/optimization.git
   cd optimization
   ```

2. 安装依赖：
    ```bash
    pip install -r requirements.txt
    ```

3. 运行示例：
    ```bash
    python Lect1/example.py
    ```
